这是一个非常典型且高质量的需求。你想要在一个复杂的场景（高定婚纱摄影、特定朝代服饰、特定环境）中，极度精确地保留人物的脸部特征。

要实现“脸部最高权重，服装中等，背景最低”并“精准抓取脸部”，我们需要结合提示词结构学（Prompt Structure）和平台特定的权重语法（Weight Syntax）。

由于你没有说明使用的具体AI绘画平台（如Midjourney, Stable Diffusion等），我将首先介绍通用的底层逻辑，然后提供针对主流平台的具体修改方案。

核心分析：AI 如何理解权重和优先级？
AI模型通常遵循以下几个原则来分配注意力：

顺序原则 (Order Matters)： 越靠前的词语，权重通常越高。模型首先奠定基础（主体），然后再添加细节（环境）。

描述密度 (Descriptive Density)： 对某事物描述的细节越多，AI就会投入越多的算力去生成它。

显式权重语法 (Explicit Syntax)： 高级玩家使用特定的符号（如 :: 或 ()）来强制分配数学上的权重。

第一部分：如何设定关键词权重 (响应你的问题 1)
要实现【脸部(最高) > 服装(中等) > 背景/风格(最低)】，我们需要重构提示词的顺序，并引入权重语法。

策略 A：利用“顺序原则”重构 (通用方案)
目前的范例 Prompt 结构比较混乱。它以一个无效的指令开头（"Extract all..." AI 实际上看不懂这句指令），然后混合了风格、镜头、服装和环境。

正确的结构应该是： [最高权重主体锚定] + [核心脸部描述] + [中等权重服装描述] + [次要权重环境与动作] + [最低权重摄影风格/技术参数]

策略 B：利用“显式权重语法” (平台特定方案)
这是最精确的方法。

Midjourney (MJ): 使用双冒号 :: 后跟数字。标准权重是 1。例如 face::2 的重要性是 background::0.5 的四倍。

Stable Diffusion (SD / WebUI): 使用括号。(word:1.2) 表示增加20%权重，[word:0.9] 表示减少10%权重，或者多层括号 (((word)))。

举例说明权重设定的思路：
假设我们要拆解你的需求：

脸部 (最高权重, 设为 Lv.3): 需要放在最前面，并且用最强的词语锁定。

服装 (中等权重, 设为 Lv.2): 描述需要详尽，但排在脸部之后。

背景/风格 (最低权重, 设为 Lv.1): 放在最后，确保它不会喧宾夺主。

第二部分：如何更精准地得到脸部特性 (响应你的问题 2)
范例 Prompt 中最大的问题在于：

开头的 "Extract all quantifiable..." 是无效废话。

结尾的 "Keep 100% of the facial features..." 放得太靠后，且过于笼统。

要精准锁脸，需要做到以下三点：

方法 1：物理特征描述锚点 (Descriptive Anchors)
不要只告诉 AI "保持原样"，要告诉它这个原样“长什么样”。你需要分析参考图的特征并写进 Prompt。

分析参考图： 鹅蛋脸 (oval face)，柔和的韩式平眉 (soft flat brows)，杏仁眼 (almond eyes)，深棕色瞳孔 (dark brown eyes)，精致挺拔的鼻子 (delicate high bridge nose)，温润的M唇 (soft M-shaped lips)，白皙无瑕肌肤 (porcelain skin)。

将这些细节加入 Prompt，能极大减少 AI 的“自由发挥”空间。

方法 2：强力锁定词 (Strong Locking phrase)
使用比 "Keep features" 更强硬的词汇，并放在句首。

好用的词： Exact facial likeness of reference image, Identical face structure, Perfect resemblance to input reference.

方法 3：视觉参考权重 (Image Prompt Weight)
这取决于平台功能。 如果是 Midjourney，使用 --iw (image weight) 参数。默认是 --iw 1，如果你想极其像照片，可以提高到 --iw 1.5 甚至 --iw 2。如果是 Stable Diffusion 使用 ControlNet，需要提高 ControlNet 的权重。

总结建议
删除废话： 永远不要在 Prompt 里写 "Extract data..." 或 "Please generate..."，直接写描述内容。

脸部前置： 将 "Keep 100% facial features" 这类要求改写成更具体的描述，并放在 Prompt 的最开头。

结构化思维： 养成 主体(脸) > 服装 > 环境 > 风格 的写作习惯。

善用工具： 如果对脸部要求极高，Stable Diffusion 配合 ControlNet (Reference 或 IP-Adapter 模型) 是目前技术上最能实现 "100%像" 的方案，远胜于单纯的 Prompt 调试。
